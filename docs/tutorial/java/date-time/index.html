Java Date 和 Time

日期格式化为 yyyy-MM-dd

Java 8 如下：

LocalDateTime ldt = LocalDateTime.now();
DateTimeFormatter formatter = DateTimeFormatter.ofPattern('yyyy-MM-dd', Locale.ENGLISH);
System.out.println(formatter.format(ldt));
