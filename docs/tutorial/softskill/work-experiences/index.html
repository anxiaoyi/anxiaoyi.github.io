工作经验

工作多年对应届生的告诫(北邮人论坛)

原文

  • 大部分宁愿吃体力的苦，也不愿吃思考的苦。体力诸如写很烂的代码，思考诸如深入调研，总结反思，举一反三等。人生是一个不断重复的过程，在什么抽象层次上重复，这或许决定了人与人之间的本质区别吧。
  • 如果对技术有追求的话，不一定非要去大厂，可以去一些口碑比较好的小厂和做互联网业务比较前沿的国企，在里面你技术绝对会飞速提升同时伴随着提升你也许会经历很多痛苦，因为大厂很多都是自研平台和基建比较好，都是自己的哪一套，再加上分工比较细，我知道很多大厂的人比如做大数据的真的就只会写Sql，如何继承开源组建，写脚本保持业务稳定都不行，国有企业真是让你从头干到尾，全链路把控，是比较累的，但是一般不会开除，所以你有时间和机会去成长
  • 如果是走技术路线的话我有两点感悟：第一点是工作带来的提升远远小于摸鱼时的自学，所以不要听信“强度大但成长快”这种鬼话；第二点是不要小看大公司的背书能力，给简历镀金还是有必要的。
  • 过来人告诉你，想通过工作进行成长，基本上你要是一个非常有心的人，对于大多数人来说，工作就是一个活口生存的工具，大多数的成长，也都不是工作主动带来的，大多是都自己的不甘和不愿随波逐流，不愿待在舒适圈带来的成长。
  • 作为工作多年的我，一个老白兔学姐，我的建议是：1. 工作方向一定要选对，这代表你天花板会足够高，未来发展会足够好，我看帖子也有人说了；2. 个人努力很重要，尤其是在前几年，不要以为工作之后就可以躺平和懈怠，毕业之后大家的差距也是从第5年左右开始体现的。不要把过多的时间浪费在无意义的兼职或者是其他无关紧要的事情上，不是说不让你娱乐，只是不要过度沉迷，时间非常宝贵，用时间去学习和沉淀，未来你会收获更多。如果你好好学习，好好撸题，后续跳槽可以多要给几K，比一个月做兼职来得轻松，这是提升收入最快的捷径。3. 人生是一个长跑，要学会判断未来的趋势，要学会利用有限的时间提高学习效率，而不是通过透支身体积攒财富。4. 前几年个人努力很重要，后续就是抱对大腿，跟行业内的牛人做有前途的事情，金钱之类的自然会来回报你，多像大佬取取经，他们的一句指导可能价值万金。

互联网各厂的普调和晋升制度(北邮人论坛)

原文

  • 字节：没有普调，每年两次优调。
  • 腾讯：每年7月普调，调薪幅度不是固定的，现在要绩效要2星以上才能普调，调多少完全看总监个人，低至几百的上至近万。每年5月和11月为晋升时间，晋升通过加薪幅度也是总监可以调，最低也得12%左右，高的有30%+的还有更夸张的
  • 阿里：勉强算是一年两次调薪，1次绩效调薪，一次升P调薪，3月和8月。连续两年3.25辞退，3.5涨薪5-15%？，3.75看情况。
  • 快手：之前一年两次调薪，一次普调，一次优调。现在没有普调了。
  • 好未来：每年四次调薪窗口，一次普调，三次优调。
  • 华为：每年两次窗口，好像没有普调，都是优调。
  • 百度：大小窗口晋升，春季普调一次，秋季优调。每年两次晋升机会，春季一次，秋季一次（优秀的才能提名），晋升通过可以调薪。另外还有啥绩优奖、攀登者啥的激励，没有固定的时间点。
  • 美团：每年两次调薪窗口，一次晋升窗口，调薪一次普调，一次优调。美团是每年六月依据去年1-12月的绩效涨薪（如果前一年入职的晚会比较吃亏）。
  • 拼多多：
  • 猿辅导：以前每年一次普调，一次优调。现在没了。
  • TP-LINK：每年一次普调，研发岗额外一次优调。
  • 大唐移动：没有普调，只有优调。
  • 滴滴：一年两次晋升。普调是每年4月底，绩效B就有。最低2%，最高20%。身边20%的例子有好几个，共同点是绩效b+orA，入职时工资低。
  • 小米：没有普调，一年一次晋升，两次优调。
  • 京东：每年一次年底调薪机会。两次晋升机会，分别是年中和年底，年中晋升面向校招生和优秀的社招，年底机会更多一些。
  • 360：
  • vivo：每年两次晋升，两次调薪，时间差不多年中和年底。
  • 网易：
  • 海康威视：每年1次普调, 1次优调，1次晋升。破格晋升具体每年几次不太确认, 至少每个组一年有一次，可能有2次。
  • 农行研发中心：没有普调，除非晋升或晋级了。
