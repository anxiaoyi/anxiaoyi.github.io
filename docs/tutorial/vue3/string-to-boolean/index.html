JavaScript String 转为 Boolean

JavaScript String 转为 Boolean

正确的方法：

let isTrue = (myValue === 'true')

注意如下写法的结果，两个结果都是 true：

let myBool = Boolean('false') // true
let myBool = !!"false"; // true
