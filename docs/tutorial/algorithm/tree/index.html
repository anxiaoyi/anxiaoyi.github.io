BST 第 K 大的数字

二叉搜索树的中序遍历为递增序列。因此这道题可转化为求 “此树的中序遍历倒序的第 k 个节点”。

class Solution {
    
    int res, k;
    
    public int kthLargest(TreeNode root, int k) {
        this.k = k;
        dfs(root);
        return res;
    }
 
    void dfs(TreeNode root) {
        if (root == null) 
            return;

        dfs(root.right);

        if (k == 0) 
            return;

        if (--k == 0) 
            res = root.val;

        dfs(root.left);
    }

}

时间复杂度：O(N)，空间复杂度：O(N)，当树退化为链表（全部是右节点）的时候，系统使用 O(N) 大小的栈空间。
