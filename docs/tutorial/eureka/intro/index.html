Netflix Eureka 简介

Eureka 干什么

Eureka 是一套服务治理框架，其包含 Server 端和 Client 端。

架构

为什么需要服务治理

A 服务需要调用 B 服务，B 服务的 URL （可能不止一个）可以通过静态配置来维护，但是随着微服务越来越复杂，静态配置就会越来越难以维护，且维护需要耗费好多人力。

组件 描述
