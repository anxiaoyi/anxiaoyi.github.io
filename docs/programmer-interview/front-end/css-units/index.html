CSS 单位

绝对单位

单位 描述
cm 厘米
mm 毫米
in 英寸
px 像素
pt points
pc picas (1pc = 12pt)

相对单位

单位 描述
em 相对于当前元素的 font-size，2em 意味着两倍 font-size 的大小
ex 相对于当前字体的 x-height
ch 相对于 ‘0’ 的宽度
rem 相对于根元素的字体大小
vw 相当于 viewport 宽度的 1%
vh 相当于 viewport 高度的 1%
vmin 相当于 viewport 较短边的 1%
vmax 相当于 viewport 较长边的 1%
% 相对于父元素
