浏览器存储

Web Storage

Web Storage 也叫 DOM Storage.

浏览器提供了两种支持 Web Storage 的对象：

  • window.localStorage：数据没有过期时间
  • window.sessionStorage：仅仅会话期间有效，关闭当前浏览器 Tab 页面的时候，数据消失

注意，这两个 Storage 对象存储的 value 只支持 String 类型，你存储其他类型，浏览器也会自动转为 String 类型存储进去。

sessionStorage.setItem('key', 'value')
localStorage.setItem('key', 'value')

存储大小

  • Cookie 允许 4KB
  • Opera 10.50+ 允许 5MB
  • Safari 8 允许 5MB
  • Firefox 34 允许 10MB
  • Chrome 允许 10MB
  • IE 允许 10MB

数据可见性

（1）LocalStorage

只有相同协议、相同 Host、相同端口，这三个都相同，才能算作是同一个 Origin。 只要两个页面处于同一 Origin ，那么存储在这一 Origin 的 LocalStorage 数据便可以自由访问。

（2）SessionStorage

而 SessionStorage 除了需要同一 Origin，还需要同一 Tab 才可以。
