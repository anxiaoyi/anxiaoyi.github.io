CSS 盒模型

作用

决定一个元素占据多大的矩形面积。

W3C 盒模型 （标准模型）

.myClass {
    box-sizing: content-box;
}

box-sizing 属性的默认值是：content-box

IE 盒模型

.myClass {
    box-sizing: border-box;
}
