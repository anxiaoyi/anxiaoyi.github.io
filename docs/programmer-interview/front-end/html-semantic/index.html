dl、dt、dd

<dl>
    <dt>火狐浏览器</dt>
    <dd>
        由 Mozilla 组织以及数百个志愿者一起开发的一款免费、开源、跨平台的 Web 浏览器。
    </dd>
    <dt>Chrome 浏览器</dt>
    <dd>
        谷歌浏览器，是一个由Google（谷歌）公司开发的开放源代码网页浏览器。
    </dd>
</dl>
